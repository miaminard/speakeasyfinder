# Speakeasy Finder

A simple webpage to find speakeasies in a specific location.

## Features

- Enter a location to search for speakeasies.
- Displays a list of fictional speakeasies for demonstration.

## How to Run

1. Start a local server:
   ```
   python -m http.server 8000
   ```
2. Open your browser and go to `http://localhost:8000`

## Files

- `index.html`: The main HTML page.
- `style.css`: Styles for the page.
- `script.js`: JavaScript for functionality.
